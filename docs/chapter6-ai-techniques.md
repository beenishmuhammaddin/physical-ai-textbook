# AI Techniques

Robots use AI to process data and make decisions. Key techniques include:

## Machine Learning

- Supervised learning for recognizing objects.
- Reinforcement learning for decision-making and navigation.

## Computer Vision

- Enable robots to see and interpret the environment.
- Object detection, tracking, and classification.

## Natural Language Processing

- Understand human commands.
- Conversational AI in humanoid robots.
